\textbf{Numerical Simulation and Benchmark Validation of Incompressible Lid-Driven Cavity Flow}
\hfill \textit{(Oct'24 - Nov'24)} 

\begin{itemize}
\item Developed a 2D incompressible Navier–Stokes solver for the lid-driven cavity using the \textbf{vorticity–streamfunction}.
\item \textbf{Validated} results against benchmark Ghia et al. data and analyzed Reynolds-number-dependent flow structures, including vortex formation and secondary eddies, using velocity profiles and streamline contours.
\end{itemize}
